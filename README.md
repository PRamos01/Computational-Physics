# Computational Physics Toolkit

A collection of C++ classes and numerical tools developed for computational physics, providing reusable methods for solving problems in physics, mathematics, and scientific computing.

---

## Repository Structure

### IntegDeriv
Numerical integration and differentiation methods for scientific computations.

### Matrix Solver
Linear algebra utilities and matrix equation solvers.

### ODESolver
Numerical methods for solving ordinary differential equations (ODEs).

### Random Walk
Simulation tools for one-dimensional (1D) random walk problems.
